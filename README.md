# powerflow

A Power Flow Calculation Project,
 currently Newton-Raphson Method and Fast Decoupled Power Flow Method implemented.

## comparison
```shell script
 time python calculation_example1.py 
```
N-R:

real    0m0.387s

user    0m0.714s

sys     0m2.225s

FDPF:

real    0m0.220s

user    0m0.380s

sys     0m1.177s

FDPF is faster than N-R method.
